# Learned-context neural networks

CS536 project

I implemented learned-context neural networks as in Sandnes et al. 2024. I extended the original approach by adding skip connections to the architecture. See writeup for details.


Reference: T. Sandnes, B. Grimstad, and O. Kolbjørnsen, “Multi-task neural networks by learned contextual inputs,” Neural Networks, vol. 179, p. 106528, Nov. 2024, doi: 10.1016/j.neunet.2024.106528.
